Holiday Labels and Lists User Documentation

Introduction

The Holiday Labels and Lists program ("HLL") is meant to be used in connection with Valley's holiday programs. It streamlines the process of creating bag and gift labels, master and donor lists, participant lists, and postcard labels from VESTA reports.

The focus of HLL's display is the "Available Label and List Files" window. The items listed there are HLL's "work product;" its reason for being.

Other items in the program's window are:

  • Controls used to select which of the available label and list files to view. You can filter the view by file type (participant list, bag labels, etc.), year, and donor, and also select whether or not to view backup files.
  • Buttons to:
    • Process VESTA reports (that is, generate more label and list files)
    • Delete backup files in order to free up disk space
    • View this help file
    • Exit the program

NOTE: HLL does not create the VESTA reports; instead, it uses reports which have already been downloaded from VESTA. Also, HLL does not delete the VESTA reports, or change them in any way -- after you have added them, they will remain wherever you left them. You can even add them again if you want.

How to Use

Before using HLL, log in to VESTA and download one or more reports. Make sure to save them as Excel files, and remember where you save them. Then, start HLL, click "Process VESTA Reports." A standard Windows file selection dialog will open, allowing you to go to the folder containing your VESTA reports and select those reports (either one at a time or multiple files at once).. HLL will then read the repors and use the information in them to create the appropriate label and list files.

Once the VESTA reports have been processed, the files generated files will show up in HLL under "Available Label and List Files." Double-click on a file and it will open up in Excel or Word as appropriate, for viewing or printing.

When you process VESTA reports, if the name of an output label or list file matches a file that already exists, the older file will be renamed. For example, if "Bag_Labels_2016_Clothing_Aht.docx" is already there, the original will be renamed "Bag_Labels_2016_Clothing_Aht.bak0001.docx," "Bag_Labels_2016_Clothing_Aht.bak0002.docx," and so on. It's possible that quite a few backup files could be generated after some time -- if so desired, the backup files can be deleted.

Where Are My Files?

HLL puts the files it generates into subfolders under the "CHRISTMAS AND HOLIDAY PROGRAMS" folder on the Valley public drive. Files are sorted by year and by type. For example, for program year 2017, the files will be located as follows:

  • Bag and Gift Labels: CHRISTMAS AND HOLIDAY PROGRAMS\2017\Christmas
  • Donor and Master Lists:  CHRISTMAS AND HOLIDAY PROGRAMS\2017\Christmas
  • Participant Lists and Postcard Labels:
    • Adopt a Family:  CHRISTMAS AND HOLIDAY PROGRAMS\2017\Christmas
    • Holiday Food Basket:  CHRISTMAS AND HOLIDAY PROGRAMS\2017\Christmas
    • Thanksgiving Basket:  CHRISTMAS AND HOLIDAY PROGRAMS\2017\Thanksgiving
  • Anything else (eg Fans/Heaters):  CHRISTMAS AND HOLIDAY PROGRAMS\2017\Other

Backup files will be placed in the same folders as the files they're backing up.

There is no provision in HLL to delete anything other than backup files. If you want to delete non-backup label or list files, or VESTA reports, you can do that by going to the storage folder via Windows Explorer and manually deleting the files.

Usage Tips

  • Instead of using many small VESTA reports, use a few larger ones. When you ask VESTA for the reports, you have the option of asking for reports for only a single donor and only a single report type (e.g., a Gift Label Report for Mike Albert Leasing). Instead of doing that, ask for a report for Toys and Clothing labels, both Gift and Bag, for all donors at once. Then tell HLL to process that one big file. It will take a while (maybe several minutes) but is a much better use of your time than processing dozens of smaller reports individually!
  • Delete the backup files every once in a while. This may help the program run a little quicker and will also free up some disk space for other uses.

Problems? Suggestions? Further Questions?

I only see files for one donor!

Use the Donor "drop-down" selecter to pick another donor. If the donor you're looking for isn't there, verify that you've processed VESTA reports pertaining to that donor. If you're unsure about a particular VESTA report, processing that report again will do no harm.

(HLL does not have a "display files for all donors" option -- that would probably not be very comfortable to use because of the clutter.)

I don't see bag/gift/donor/master files for a donor!

It may be that that particular donor is not participating in that particular program.
